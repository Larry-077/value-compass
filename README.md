# 🧭 价值罗盘 ValueCompass

> 一门「伦理设计」课程的项目。本系统把抽象的 AI 伦理价值**编码**为可编辑的《价值手册》，并将其构造成**奖励信号**驱动大模型对齐，让"价值对齐"变得**可编辑、可运行、可度量**。主题落在 AI 伦理 / 算法偏见 / 透明性 / 负责任 AI。

核心做了三件事：
1. **编码价值**——以 HHH 为骨架、逐条标注权威出处（UDHR/法规）、配锚定式评分细则的《价值手册》；
2. **对齐与度量**——自我批评-修订、Best-of-N（推理期 RLHF）、从人类偏好训练奖励模型，并量化对齐前后效果；
3. **构造难题**——现代模型在普通题上已对齐、测不出问题，故用加压技术构造能让模型"翻车"的困难数据集（核心贡献）。

## 快速开始

```bash
npm install && npm run dev      # 打开 http://localhost:5173
```
**无需配置 API Key** 即可看到作者用真实模型跑出的结果（已内置）；想亲自实时运行，复制 `.env.example` 为 `.env` 填入一个免费智谱 GLM 令牌即可。
